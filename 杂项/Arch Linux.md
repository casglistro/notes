# 1. OBS 无法录屏
使用的是 `Hyprland` 环境，改为使用 `xdg-desktop-portal-wlr` 和 `wlrobs`。
```
yay -S wlrobs xdg-desktop-portal-wlr
```
然后使用 `scpy` 源就可以录屏了。
